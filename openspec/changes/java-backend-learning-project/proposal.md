## 为什么

作为一名拥有10年经验的前端开发工程师，我需要转型到Java后端开发，以适应AI时代的开发格局变化，AI使得全栈开发能力变得更加广泛。这个项目将提供一个结构化的学习路径和实践经验，帮助我掌握生产级的Java后端开发技能。

## 变更内容

- 使用Spring Boot框架从零开始创建完整的Java后端学习项目
- 建立涵盖Java核心概念、后端架构和Spring生态系统的基础知识路径
- 构建包含数据库集成、身份认证和常见企业级模式的RESTful API项目
- 提供测试、部署和最佳实践的实践模块
- 包含学习资源、代码示例和渐进式练习

## 能力模块

### 新增能力
- `java-fundamentals`: 后端开发必需的Java核心概念（面向对象、集合、流式处理、并发）
- `spring-boot-setup`: 项目初始化、依赖管理和Spring Boot应用程序结构
- `rest-api-development`: RESTful API设计、控制器、请求/响应处理和HTTP方法
- `database-integration`: JPA/Hibernate、数据库连接、CRUD操作和事务管理
- `authentication-security`: Spring Security、JWT身份认证、授权和安全最佳实践
- `testing-strategy`: 单元测试、集成测试、Mock和测试驱动开发
- `error-handling`: 异常处理、数据验证和标准化错误响应
- `logging-monitoring`: 应用程序日志记录、监控和调试技术
- `deployment-basics`: 构建、打包和部署Java应用程序
