# Java面向对象编程 (OOP)

## 前言

作为前端开发者，你可能已经熟悉JavaScript/TypeScript的类。Java是纯面向对象语言，一切皆对象。让我们深入学习Java的OOP特性。

## OOP四大特性

1. **封装 (Encapsulation)**: 隐藏内部实现细节
2. **继承 (Inheritance)**: 代码复用和扩展
3. **多态 (Polymorphism)**: 同一接口，不同实现
4. **抽象 (Abstraction)**: 定义规范，隐藏复杂性

## 1. 类和对象

### 基本类定义

```java
// Person.java
public class Person {
    // 字段（属性）
    private String name;
    private int age;
    
    // 构造函数
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // 方法
    public void introduce() {
        System.out.println("我是" + name + "，今年" + age + "岁");
    }
    
    // Getter和Setter
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    public int getAge() {
        return age;
    }
    
    public void setAge(int age) {
        if (age > 0) {  // 数据验证
            this.age = age;
        }
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        Person person = new Person("张三", 25);
        person.introduce();
        
        person.setAge(26);
        System.out.println(person.getName() + "现在" + person.getAge() + "岁");
    }
}
```

## 2. 访问修饰符

Java有四种访问级别：

| 修饰符 | 类内部 | 同包 | 子类 | 任何地方 |
|--------|--------|------|------|----------|
| `private` | ✅ | ❌ | ❌ | ❌ |
| (default) | ✅ | ✅ | ❌ | ❌ |
| `protected` | ✅ | ✅ | ✅ | ❌ |
| `public` | ✅ | ✅ | ✅ | ✅ |

```java
public class AccessExample {
    private String privateField = "私有";       // 只能在类内部访问
    String defaultField = "默认";              // 同包可访问
    protected String protectedField = "保护";   // 子类可访问
    public String publicField = "公共";        // 任何地方可访问
    
    private void privateMethod() {
        System.out.println("私有方法");
    }
    
    public void publicMethod() {
        System.out.println("公共方法");
        privateMethod();  // 内部可以调用private方法
    }
}
```

## 3. 构造函数

### 默认构造函数

```java
public class Book {
    private String title;
    private String author;
    
    // 无参构造函数
    public Book() {
        this.title = "未命名";
        this.author = "未知";
    }
    
    // 有参构造函数

    }

Book book3 = new Book("设计模式");
```

## 4. 继承 (Inheritance)

### 基本继承

```java
// 父类
public class Animal {
    protected String name;
    protected int age;
    
    public Animal(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    public void eat() {
        System.out.println(name + "在吃东西");
    }
    
    public void sleep() {
        System.out.println(name + "在睡觉");
    }
}

// 子类
public class Dog extends Animal {
    private String breed;  // 品种
    
    public Dog(String name, int age, String breed) {
        super(name, age);  // 调用父类构造函数
        this.breed = breed;
    }
    
    // 方法重写 (Override)
    @Override
    public void eat() {
        System.out.println(name + "正在吃狗粮");
    }
    
    // 新方法
    public void bark() {
        System.out.println(name + "在汪汪叫");
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        Dog dog = new Dog("旺财", 3, "金毛");
        dog.eat();    // 调用重写的方法
        dog.sleep();  // 调用继承的方法
        dog.bark();   // 调用自己的方法
    }
}
```

### 方法重写规则

```java
public class Parent {
    // 方法重写的规则：
    // 1. 方法签名必须完全相同
    // 2. 返回类型相同或是子类型（协变返回类型）
    // 3. 访问修饰符不能更严格
    // 4. 不能抛出新的或更广的检查异常
    
    protected Number getValue() {
        return 100;
    }
}

public class Child extends Parent {
    // ✅ 正确：协变返回类型
    @Override
    protected Integer getValue() {  // Integer是Number的子类
        return 200;
    }
    
    // ❌ 错误：不能降低访问级别
    // private Number getValue() { ... }
    
    // ❌ 错误：不能改变参数列表（这是重载，不是重写）
    // public Number getValue(int x) { ... }
}
```

## 5. 多态 (Polymorphism)

### 编译时多态（方法重载）

```java
public class Calculator {
    // 方法重载：同名方法，不同参数
    public int add(int a, int b) {
        return a + b;
    }
    
    public double add(double a, double b) {
        return a + b;
    }
    
    public int add(int a, int b, int c) {
        return a + b + c;
    }
    
    public String add(String a, String b) {
        return a + b;
    }
}
```

### 运行时多态

```java
public class Shape {
    public void draw() {
        System.out.println("绘制形状");
    }
    
    public double area() {
        return 0;
    }
}

public class Circle extends Shape {
    private double radius;
    
    public Circle(double radius) {
        this.radius = radius;
    }
    
    @Override
    public void draw() {
        System.out.println("绘制圆形");
    }
    
    @Override
    public double area() {
        return Math.PI * radius * radius;
    }
}

public class Rectangle extends Shape {
    private double width;
    private double height;
    
    public Rectangle(double width, double height) {
        this.width = width;
        this.height = height;
    }
    
    @Override
    public void draw() {
        System.out.println("绘制矩形");
    }
    
    @Override
    public double area() {
        return width * height;
    }
}

// 多态的威力
public class Main {
    public static void main(String[] args) {
        // 父类引用指向子类对象
        Shape shape1 = new Circle(5);
        Shape shape2 = new Rectangle(4, 6);
        
        // 运行时决定调用哪个方法
        shape1.draw();  // 输出: 绘制圆形
        shape2.draw();  // 输出: 绘制矩形
        
        // 数组中存储不同类型的对象
        Shape[] shapes = {
            new Circle(5),
            new Rectangle(4, 6),
            new Circle(3)
        };
        
        // 统一处理
        for (Shape shape : shapes) {
            shape.draw();
            System.out.println("面积: " + shape.area());
        }
    }
}
```

## 6. 抽象类 (Abstract Class)

```java
// 抽象类不能被实例化
public abstract class Vehicle {
    protected String brand;
    protected int year;
    
    public Vehicle(String brand, int year) {
        this.brand = brand;
        this.year = year;
    }
    
    // 抽象方法（没有实现）
    public abstract void start();
    public abstract void stop();
    
    // 具体方法
    public void displayInfo() {
        System.out.println("品牌: " + brand + ", 年份: " + year);
    }
}

public class Car extends Vehicle {
    private int doors;
    
    public Car(String brand, int year, int doors) {
        super(brand, year);
        this.doors = doors;
    }
    
    // 必须实现所有抽象方法
    @Override
    public void start() {
        System.out.println("汽车启动：插入钥匙，转动");
    }
    
    @Override
    public void stop() {
        System.out.println("汽车停止：踩刹车");
    }
}

public class Motorcycle extends Vehicle {
    public Motorcycle(String brand, int year) {
        super(brand, year);
    }
    
    @Override
    public void start() {
        System.out.println("摩托车启动：按下启动按钮");
    }
    
    @Override
    public void stop() {
        System.out.println("摩托车停止：拉手刹");
    }
}
```

## 7. 接口 (Interface)

### 基本接口

```java
// 接口定义规范
public interface Flyable {
    // 接口中的方法默认是public abstract
    void fly();
    void land();
    
    // Java 8+: 默认方法
    default void glide() {
        System.out.println("滑翔中...");
    }
    
    // Java 8+: 静态方法
    static void checkWeather() {
        System.out.println("检查天气状况");
    }
}

// 实现接口
public class Airplane implements Flyable {
    @Override
    public void fly() {
        System.out.println("飞机起飞");
    }
    
    @Override
    public void land() {
        System.out.println("飞机降落");
    }
}

public class Bird implements Flyable {
    @Override
    public void fly() {
        System.out.println("鸟儿飞翔");
    }
    
    @Override
    public void land() {
        System.out.println("鸟儿着陆");
    }
}
```

### 多接口实现

```java
public interface Swimmable {
    void swim();
}

public interface Walkable {
    void walk();
}

// 一个类可以实现多个接口
public class Duck implements Flyable, Swimmable, Walkable {
    @Override
    public void fly() {
        System.out.println("鸭子飞行");
    }
    
    @Override
    public void land() {
        System.out.println("鸭子降落");
    }
    
    @Override
    public void swim() {
        System.out.println("鸭子游泳");
    }
    
    @Override
    public void walk() {
        System.out.println("鸭子行走");
    }
}
```

### 接口 vs 抽象类

| 特性 | 接口 (Interface) | 抽象类 (Abstract Class) |
|------|------------------|------------------------|
| 继承 | 可实现多个接口 | 只能继承一个类 |
| 构造函数 | ❌ 不能有 | ✅ 可以有 |
| 字段 | 只能有常量 | 可以有任何字段 |
| 方法 | 默认public abstract | 可以有任何访问修饰符 |
| 使用场景 | 定义能力/契约 | 相关类的共同基础 |

**选择建议**:
- 表达"是什么" → 用抽象类（Car is a Vehicle）
- 表达"能做什么" → 用接口（Duck can fly, swim, walk）

## 8. 静态成员

```java
public class Counter {
    // 静态变量（类变量）- 所有实例共享
    private static int count = 0;
    
    // 实例变量
    private int id;
    
    public Counter() {
        count++;  // 每创建一个实例，count加1
        this.id = count;
    }
    
    // 静态方法
    public static int getCount() {
        return count;
        // 注意：静态方法中不能访问实例变量
        // return this.id;  // ❌ 编译错误
    }
    
    public int getId() {
        return id;
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        System.out.println("初始计数: " + Counter.getCount());  // 0
        
        Counter c1 = new Counter();
        Counter c2 = new Counter();
        Counter c3 = new Counter();
        
        System.out.println("当前计数: " + Counter.getCount());  // 3
        System.out.println("c1 ID: " + c1.getId());  // 1
        System.out.println("c2 ID: " + c2.getId());  // 2
        System.out.println("c3 ID: " + c3.getId());  // 3
    }
}
```

## 9. 内部类

### 成员内部类

```java
public class Outer {
    private String outerField = "外部类字段";
    
    // 成员内部类
    public class Inner {
        private String innerField = "内部类字段";
        
        public void display() {
            System.out.println("外部类: " + outerField);  // 可访问外部类成员
            System.out.println("内部类: " + innerField);
        }
    }
    
    public void test() {
        Inner inner = new Inner();
        inner.display();
    }
}

// 使用
Outer outer = new Outer();
Outer.Inner inner = outer.new Inner();
inner.display();
```

### 静态内部类

```java
public class Outer {
    private static String staticField = "静态字段";
    private String instanceField = "实例字段";
    
    // 静态内部类
    public static class StaticInner {
        public void display() {
            System.out.println(staticField);  // ✅ 可以访问
            // System.out.println(instanceField);  // ❌ 不能访问实例字段
        }
    }
}

// 使用
Outer.StaticInner inner = new Outer.StaticInner();
inner.display();
```

### 匿名内部类

```java
public interface Greeting {
    void greet();
}

public class Main {
    public static void main(String[] args) {
        // 匿名内部类
        Greeting greeting = new Greeting() {
            @Override
            public void greet() {
                System.out.println("Hello from anonymous class!");
            }
        };
        greeting.greet();
        
        // Java 8+: Lambda表达式（更简洁）
        Greeting greeting2 = () -> System.out.println("Hello from lambda!");
        greeting2.greet();
    }
}
```

## 10. 枚举 (Enum)

```java
// 简单枚举
public enum DayOfWeek {
    MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY, SUNDAY
}

// 带字段和方法的枚举
public enum Size {
    SMALL("S", 1),
    MEDIUM("M", 2),
    LARGE("L", 3),
    XLARGE("XL", 4);
    
    private final String abbreviation;
    private final int size;
    
    Size(String abbreviation, int size) {
        this.abbreviation = abbreviation;
        this.size = size;
    }
    
    public String getAbbreviation() {
        return abbreviation;
    }
    
    public int getSize() {
        return size;
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        // 枚举比较
        DayOfWeek today = DayOfWeek.MONDAY;
        if (today == DayOfWeek.MONDAY) {
            System.out.println("今天是星期一");
        }
        
        // 遍历枚举
        for (DayOfWeek day : DayOfWeek.values()) {
            System.out.println(day);
        }
        
        // 使用枚举方法
        Size mySize = Size.LARGE;
        System.out.println("缩写: " + mySize.getAbbreviation());
        System.out.println("尺码: " + mySize.getSize());
    }
}
```
